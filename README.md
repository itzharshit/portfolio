# Harshit Shrivastav Portfolio 🚀

**A modern, interactive portfolio website built with Astro.js**

🌐 **Live:** [harshitshrivastav.eu.org](https://harshitshrivastav.eu.org/)

---

## ✨ Features

- 🪐 **Saturn Cursor** - Animated Saturn-like cursor with revolving rings
- 💥 **Big Bang Effect** - Spectacular explosion animation on 5 clicks
- 📱 **Fully Responsive** - Optimized for all screen sizes (mobile, tablet, desktop)
- 🎨 **Dark Theme** - Beautiful dark UI with golden accents
- 🔊 **Sound Effects** - Explosion sound on big bang
- 🌍 **SEO Optimized** - Complete meta tags, structured data, sitemap
- 🤖 **AI/LLM Ready** - llms.txt and llms-full.txt for AI discovery
- ♿ **Accessible** - ARIA labels, keyboard navigation, skip links
- ⚡ **Fast** - Built with Astro for optimal performance

---

## 🛠️ Tech Stack

- **Framework:** Astro.js
- **Styling:** Custom CSS with CSS Variables
- **Animations:** Vanilla JavaScript + CSS Keyframes
- **Audio:** HTML5 Audio API
- **Hosting:** Static deployment ready

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/itzharshit/portfolio.git
cd portfolio/portfolio-astro

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
# Build static site
npm run build

# Preview production build
npm run preview
```

### Deploy

The `dist/` folder contains the production-ready static files. Deploy to any static hosting:

- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- Any static host

## 🎯 Features Guide

### Saturn Cursor
- Appears next to navigation logo on page load
- Follows mouse movement
- Hidden on touch devices for better UX

### Big Bang Effect
- Click 5 times rapidly in the same area
- Triggers massive explosion with:
  - Flash bang
  - Shockwave rings
  - Particle debris
  - Spark embers
  - Fire trails
  - Screen shake effect
  - Explosion sound

### Mobile Responsive
- 3 breakpoints: 900px, 600px, 400px
- Optimized layouts for each screen size
- Touch-friendly interactions
- Cursor hidden on touch devices
